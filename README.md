## Installation

BEKSpeechToText is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```cocoapod
pod 'BEKSpeechToText', :git => 'https://github.com/behrad-kzm/BEKSpeechToText.git'
```
